## Project Overview
End-to-end design and implementation of a secure enterprise network using Cisco Packet Tracer.
Built with layered trust zones, centralized security controls, and validated through real traffic flow testing.

## Key Features
- VLAN segmentation (HR, Engineering, Guest, Voice, Servers, Management)
- Cisco ASA stateful firewall with DMZ architecture and Static NAT
- WPA2-Enterprise wireless with RADIUS authentication
- TACACS+ for centralized network device administration
- ACL-based traffic enforcement and guest network isolation
- Centralized Syslog server and NTP time synchronization
- Full testing and validation of allowed and blocked traffic scenarios
